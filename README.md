# Pet Care Booking Platform (UK Market)

## Project Overview
The Pet Care Booking Platform is a comprehensive, dual-sided web application designed to connect pet owners with vetted service providers (groomers, walkers, sitters, and vets) across the UK. 

## Key Features
* **Role-Based Access:** Distinct, highly detailed dashboards for Pet Owners and Service Providers.
* **Deep Pet Profiles:** 60-point data collection for pets, including medical history, behavioral triggers, and vet details.
* **Booking Negotiation Workflow:** Providers can review a pet's deep profile before accepting a job. Owners then propose a time, which the provider approves.
* **In-App Communication:** Real-time, secure messaging linked directly to specific bookings.
* **Dual-Sided Feedback:** A community rating system where owners review providers, and providers review owners.

## Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Python, Django REST Framework
* **Database:** SQLite (Development)
